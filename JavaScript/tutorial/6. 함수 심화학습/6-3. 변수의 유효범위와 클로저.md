# 6. 함수 심화학습
## 6-3. 변수의 유효범위와 클로저
### 변수의 유효범위 예시
- **코드 블록**   
    코드 블록 `{...}` 안에서 선언한 변수는 블록 안에서만 사용 가능함
```javascript
{
    let message = "안녕하세요.";
    alert(message); // 안녕하세요.
}
alert(message); // ReferenceError: message is not defined


if (true) {
    let phrase = "안녕하세요!";
    alert(phrase); // 안녕하세요!
}
alert(phrase); // ReferenceError: phrase is not defined


for (let i = 0; i < 3; i++) {
    alert(i); // 0, 1, 2 출력
}
alert(i); // ReferenceError: i is not defined
```
